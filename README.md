# Finpro

This is a cool Angular 2 application that lets you create watchlists of your favourite stocks and track their prices with nice graphs! 

Developed using Angular 2 RC6. 

Also uses the following:

            * Google Finance API (for stock quotes)
            * angular-cli
            * angular2-highcharts
            * ng2-bootstrap
            * RxJS
            * HTML5 Local Storage
